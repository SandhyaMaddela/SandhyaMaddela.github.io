Training

During my enrollment in the Java Developer Program at NIIT Institute from June 2023 to November 2023, I immersed myself in an intensive curriculum that honed my skills in Java programming and SQL database management. Through hands-on projects and assignments, I applied theoretical knowledge to practical scenarios, solidifying my understanding of Java's core principles, particularly Object-Oriented Programming (OOP) concepts. This comprehensive training equipped me with the expertise needed to develop robust and efficient database-driven applications.

Additionally, I undertook a pivotal project titled "Digital Jukebox," where I played a central role in its development and implementation. This project aimed to create an intuitive digital music platform for users to enjoy their favorite songs seamlessly. My contributions encompassed various key aspects:

  • User Interface Design: I crafted an intuitive user interface that streamlined navigation and song selection, ensuring a user-friendly experience
  • Song and Playlist Management: Utilizing Java programming principles, I implemented robust song and playlist management systems. This allowed users to efficiently search for songs and create personalized playlists tailored to their preferences.
  • Database Functionality: Leveraging my SQL database management skills, I developed the database functionality for storing song data and user-created playlists. This involved implementing features for efficient data retrieval, modification, and seamless integration with the application.
